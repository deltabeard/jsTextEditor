Notes:
The text box that changes the text colour using the user input can accept:

  • Hexadecimal colors, eg. "#FF0000"
  • RGB colors, eg. "rgb(255,0,0)"
  • RGBA colors, eg. "rgba(255, 0, 0, 0.8)"
  • HSL colors, eg. "hsl(240,100%, 50%)"
  • HSLA colors, eg. "hsla(0,100%,50%,0.8)"
  • Predefined/Cross-browser color names, eg. "Red"
